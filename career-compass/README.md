# Mentor Focus Board

A static Vercel-ready website for mentor meetings.

## What it does
- shows compact career pathways with icons
- shows core skill foundations as draggable chips
- lets a mentor drag items into four priority lanes
- stores progress in localStorage
- exports the board as JSON
- includes a lightweight Three.js animated background

## Deploy to Vercel
1. Upload this folder to GitHub or drag it into Vercel.
2. Framework preset: `Other`
3. Build command: leave empty
4. Output directory: `.`

## Local preview
Open `index.html` directly, or run a simple static server.
